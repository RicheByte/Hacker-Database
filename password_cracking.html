Common Cracking Commands


# John the Ripper
john --wordlist=wordlist.txt hash.txt

# Hashcat (MD5)
hashcat -m 0 hash.txt wordlist.txt

# Hydra (SSH brute-force)
hydra -l username -P wordlist.txt target.example.com ssh

# Ncrack (parallel login cracking)
ncrack -v -u user -P wordlist.txt target.example.com

# Medusa (SSH brute-force)
medusa -h target.example.com -U user.txt -P pass.txt -M ssh

# GPU-powered Hashcat
oclHashcat -m 0 hash.txt wordlist.txt

# RainbowCrack (using rainbow tables)
rainbowcrack -l rainbow_table.rt hash.txt

# Generate custom wordlist using Cewl
cewl -w wordlist.txt http://target.example.com
Tips

Best Practices

  • Use multiple wordlists for comprehensive testing
  • Combine dictionary attacks with rule-based mutations
  • Leverage GPU acceleration for faster cracking
  • Always obtain proper authorization before testing
  • Use masked attacks for known password patterns
